Frequently Asked Questions

Everything you need to know about the SecureAuth Bootcamp

Can I switch paths after enrolling?
Path enrollment is designed to be permanent to encourage focus and commitment. If you need to switch, please contact adrianmange00@gmail.com to request a path change.
Do I need previous cybersecurity experience?
The Student Path starts from fundamentals and requires no prior experience. The Developer Path assumes basic programming knowledge and is best suited for those with some development background.
How does the XP and streak system work?
You earn XP by completing lessons, quizzes, and labs. Your streak tracks consecutive days of activity. Both are displayed on your dashboard and contribute to your rank progression.
I'm a lecturer — how do I access the platform?
Lecturer accounts are created by administrators. If you need lecturer access, please contact adrianmange00@gmail.com or visit the Academy directly.
Is this free?
Yes, the SecureAuth Bootcamp is completely free for all registered users. Create an account and start learning immediately.
